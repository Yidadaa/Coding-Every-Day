# GSOC2020 JdeRobot CPP Challenge: Labyrinth

Find longest pathway using BFS.

## How to Use
Build the source code and run:
```bash
mkdir build && cd build
cmake .. && make
./labyrinth.cpp ../data/2.txt
```

Example output for `2.txt`:
```
9
##.##0#
##.#21#
####34#
##8765#
#######
```
